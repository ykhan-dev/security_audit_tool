## âš¡ Features

- SSH service status check (macOS/Linux)  
- Top 10 running services check  
- TCP port scanning (supports custom ports and ranges, e.g., 20-25 80 443)  
- Color-coded terminal output for clear visualization

## ðŸš€ Usage

Run the main audit script:

```bash
python3 app/main_audit.py [--target <IP>] [--ports 22 80 443 1000-1010] [--output-dir <folder>]
```

**Arguments:**

- `--target` : Target IP to scan (default: 127.0.0.1)  
- `--ports`  : Ports to scan (supports ranges, default: 22 80 443 5000 5432)  
- `--output-dir` : Directory to save logs and reports (default: current folder)

**Example:**

```bash
python3 app/main_audit.py --target 192.168.1.10 --ports 22 80 443 1000-1010 --output-dir reports/
```

**Output:**

- Terminal displays color-coded scan results  
- Log file: `logs/security_audit_log_<timestamp>.txt`  
- Report file: `reports/security_audit_report_<timestamp>.txt`
